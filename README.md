# AI Shop Platform

Платформа для создания интернет-магазинов с ИИ-помощником

## Функциональность

- Создание интернет-магазинов через ИИ-помощника (Hugging Face API)
- Генерация описания магазина и категорий товаров с помощью нейросети
- Управление товарами, категориями, брендами и новостями
- Публичная витрина для покупателей
- Админ-панель Django
- Несколько магазинов у одного пользователя
- CRUD операции для всех сущностей

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ZeroUzer/ai.git
cd ai

### 2. Создание виртуального окружения
python -m venv venv
venv\Scripts\activate

### 3. Установка зависимостей
pip install -r requirements.txt


### 4. Применение миграций
python manage.py migrate

### 5. Создание суперпользователя
python manage.py createsuperuser

### 6. Запуск сервера
python manage.py runserver


### Как пользоваться
Создание магазина через ИИ
Перейдите на страницу /ai/create/

Введите название магазина и описание (что продаете)

Нажмите "Сгенерировать магазин с ИИ"

ИИ создаст описание и предложит категории

Подтвердите создание

Управление магазином
Перейдите в /shops/my/ - список ваших магазинов

Нажмите "Управлять" у нужного магазина

Добавляйте товары, категории, новости через админку

Ссылка на витрину: /shop/ваш-магазин/

Админка
Доступна по адресу /admin/

Управление всеми товарами, категориями, брендами, новостями
